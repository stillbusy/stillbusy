# SurviveSpace 🏠

The ultimate survival system for independent living. Manage expenses, chores, and emergencies in shared accommodations.

## 🚀 Features

- **Dashboard:** Real-time overview of group spending, active chores, and shopping needs.
- **Split Expenses:** Track group bills, see who owes what, and settle up effortlessly.
- **Chore Divider:** Fairly distribute household tasks with status tracking.
- **Shopping List:** Collaborative list for groceries and household essentials.
- **Survival Mode:** Budget meal planning and inventory management for the "broke days."
- **Emergency Mode:** Quick access to local help and emergency contacts.
- **Real-time Sync:** Powered by Socket.io for instant updates across all roommates.

## 🛠️ Tech Stack

- **Frontend:** React, TypeScript, Tailwind CSS, Lucide Icons, Motion (Animations)
- **Backend:** Node.js, Express, Socket.io
- **Build Tool:** Vite

## 📦 Installation

1. Clone the repository:
   ```bash
   git clone https://github.com/stillbusy/survivespace.git
   ```
2. Install dependencies:
   ```bash
   npm install
   ```
3. Start the development server:
   ```bash
   npm run dev
   ```
